# lemnos

> A personal collection of tools I actually use.

Built with [Quarto](https://quarto.org). No fluff.

## What's inside

Curated notes and configs for tools like `yt-dlp`, `Typora`, and others — focused on setup, key options, and the parts worth remembering.

## Structure

```
lemnos/
├── tools/        # one page per tool
├── _quarto.yml   # site config
└── index.qmd     # home
```

## Usage

```bash
quarto preview
quarto render
```
